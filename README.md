# [cestina.top](https://cestina.top)

Tahák do češtiny. Jednoduché vysvětlení nejčastějších chyb v českém pravopisu.

Projekt [SmirkHat.org](https://smirkhat.org).

## Témata

mně/mě, jí/ji, mě/mně ve slovech, tip/typ, shlédnout/zhlédnout, výjimka, holt/hold, jsi/si, sebou/s sebou, s/z předložky, čárky, sloveso být

## Sdílení

Každé téma má vlastní URL s embedem pro Discord a sociální sítě:

```
cestina.top/mne-me
cestina.top/ji-ji
cestina.top/tip-typ
cestina.top/carky
...
```

## Tech stack

- Čisté HTML + CSS + minimální JS, žádný build
- GitHub Pages
- Geist font (Vercel)
- Dark-first design
